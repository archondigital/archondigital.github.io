Archon Digital's 5th Incarnation

{% include entry-meta.html %} {% include snippet-disclaimer-old-post.html %}

Archon Digital, the 5th version finally launches after months of having shelved the project, this grand scheme of World Domination, due to busy schedules and a some other minor complications.

Moving out of the confines of a traditional blog's layout has always been what I wanted to achieve here on Archon Digital since 2007. These latest changes now allow me to bend an article's layout and move out of the structured form yet retain the structure should I choose to be lazy and just blog away normally. I've overhauled nearly all of the older articles on the website for code problems and updated invalid custom fields previously used and implemented some new ones.

The Design

After months of thinking about how a new look for version 5.0 would stand out and be an actual improvement from the previous, this is what I have come up with. A fixed/fluid width layout with special blogazine-like articles apart from the regular blog articles and a better and speedier loading design. After testing extensively for this past few days, I can say this has got to be the most complicated yet fastest version of the site since version 2.0 and all of this of was really for me to get motivated and update Archon Digital's content regularly.

Custom WordPress Loops, Category Switches and Custom Fields

Major changes have been made with regards to how Archon Digital organizes featured posts, studio articles and portfolio entries. Category switches are out, the WordPress loop handling special posts now filters queries using custom fields. I would prefer to discuss this in detail and with code samples on another post I am working on.

Fluid and Fixed width

While some elements of this site are styled to perform better on wide screens by having a more fluid layout, majority of the structure is still on a fixed-width structure making use of a 12-column, 960px grid. Also taking advantage of the extra screen real estate to get all the social media clutter out of the way by having them tucked nicely in a jQuery slide pane in the footer.

Thematic, Child Themes and WordPress Theme Frameworks

Previous versions of Archon Digital have been coded as WordPress themes from scratch or by using blank WordPress themes like Vorlage in version 3.0 and Starkers in version 4.0. For the fifth version, a lot of time was spent deciding which framework to use. Working on theme frameworks has been a lot easier from a development point of view. I've come across a lot of frontend frameworks and for the past months have been ended up using Thematic for most projects. Archon Digital Version 5.0 is a child theme of Thematic.

Moving Forward

And now it can be said, World domination is at hand.
